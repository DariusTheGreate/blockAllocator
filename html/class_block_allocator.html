Detailed Description

Custom allocator based on malloc/free. Allocates _blockCount number of chunks with size of _blockSize. Stores list of free chunks, and gives free one on allocate(). Put one back on deallocate(). Free list goes from the end to start. Not STL compatible yet:( Stores mutex for synchronization. Possible improvement - use lock-free list for free chunks.
